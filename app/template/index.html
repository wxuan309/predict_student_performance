What does the project do?

Due to the advancement of technology and availability of data, it makes possible for educators and policymakers to understand students’ behavior, keep track of their progress, and further support their learning experiences. To this end, we aim to explore the factors that affect students’ academic performance and further predict their achievement.

Exploration


Data Overview

The datasets are retrieved from UCI machine learning repository, containing 1044 observations and 32 columns in total. The combined two datasets are students’ performance in Mathematics and Portuguese language. It includes demographic, personal, social, and parental features that may play a role in their achievement. Each observations represent a student and includes both numeric and categorical data. It has no missing values and three response variables: G1, G2, and G3. It represents a student’s performance for term 1, term 2, and term 3.

Machine Learning


We will be using Logistic Regression as the baseline model and following random forest and XGBoost in predicting students’ performance for term 1, term 2, and term3. Along with a Grid search to tune the hyperparameters, parameter details for each model will be provided below. As for evaluating the performance of models, we will be using the confusion matrix.

Logistic Regression: Explanation

Logistic Regression is one of the simplest and basic machine learning models for binary classification with great interpretability. It correlates all features with the response variable through a linear relationship. To this end, it is common that it serves as a baseline model for most machine learning projects.

Linear Regression: Results

Random Forest: Explanation

Random Forest is an ensemble method that combines and implements the concepts of bagging and Decision Tree, where different bootstrap samples from the training data are fitted with Decision Trees. At each node, random feature subsets are derived from deciding upon the optimal split to reduce variance and improve accuracy.

Random Forest: Results

XGBoost: Explanation

Xgboost is also an ensemble that combines the outputs from individual trees. It uses boosting by combining weak learners sequentially so that each new tree corrects the errors of the previous one, and this process continues until no further improvements can be made.

XGBoost: Results

Conclusion


Surprisingly, logistic regression as a baseline model is the best model compared to random forest and XGBoost. It has an overall higher accuracy and f1 score in predicting the three response variables. Also, as each model’s performance enhances from terms 1, 2, to 3, it is reasonable to assume that a student’s score in previous terms is an important indicator of future performances. There is a correlation between students’ performance between term 1, term 2, and term3.

So what?

As we gained an understanding that students’ performance in past terms serves as an indicator of future performances, it is crucial for educators to keep track and support those who fall behind.
